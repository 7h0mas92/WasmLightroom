# 📸 WasmLightroom

Éditeur photo en ligne avec filtres WebAssembly (C++/Emscripten), backend PHP, et base de données MySQL.

## 🚀 Lancement rapide

```bash
# Cloner le projet
git clone https://github.com/votre-user/WasmLightroom.git
cd WasmLightroom

# Lancer avec Docker Compose
docker compose up --build -d

# Accéder à l'application
open http://localhost:8080
